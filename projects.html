// project_overview

Developed an interactive Rule-Based College Information Chatbot that answers student queries related to college facilities, courses, departments, admissions, and student services through predefined rule-based logic. Built using Python, Flask, HTML, CSS, JavaScript, and SQLite — delivering a complete full-stack end-to-end solution.

// key_features
Interactive chatbot that answers student queries through predefined rule-based logic.
Built using Python, Flask, HTML, CSS, JavaScript, and SQLite — a full-stack solution.
Provides quick and efficient responses through intelligent predefined rules and seamless database integration.
Deployed locally with a clean, responsive web interface for students to interact with in real time.
// what_i_built
Designed and developed the complete chatbot interface and backend functionality.
Implemented rule-based logic engine to efficiently match and respond to user queries.
Integrated SQLite database for structured storage and retrieval of college information.
Developed RESTful Flask routes to connect the frontend chat interface with the backend engine.
Tested, debugged, and iteratively improved chatbot response accuracy and user experience.

Successfully delivered a fully functional chatbot capable of handling a wide range of college-related queries.
Gained hands-on full-stack development experience — from database design to UI rendering.
Strengthened understanding of Flask routing, SQLite queries, and rule-based AI systems.
